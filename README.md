# Крестики-Нолики Онлайн

Игра для двоих на Next.js. Состояние партии хранится в Redis (Upstash),
поэтому в неё можно играть с другом из любой точки мира — каждый открывает
сайт у себя, один создаёт комнату, другой подключается по коду.

## Деплой на Vercel — шаг за шагом

1. **Залей код на GitHub**
   Создай новый репозиторий на github.com и запушь туда все файлы из этой папки.

2. **Создай бесплатную базу Redis**
   Зайди на vercel.com → открой свой проект (или импортируй репозиторий
   с GitHub, см. шаг 3) → вкладка **Storage** → **Create Database** →
   выбери **Upstash** (Redis) → следуй подсказкам, создай базу.
   Vercel сам подключит её к проекту и добавит нужные переменные окружения
   — руками их вводить не нужно.

3. **Импортируй проект в Vercel**
   На vercel.com нажми **Add New → Project**, выбери свой GitHub-репозиторий,
   нажми **Deploy**. Framework Preset определится автоматически как Next.js.

4. **Привяжи базу к проекту** (если не сделал это на шаге 2)
   В настройках проекта → **Storage** → **Connect Store** → выбери
   созданную Upstash-базу. После этого нажми **Redeploy**, чтобы переменные
   окружения подхватились.

5. **Готово**
   Vercel выдаст ссылку вида `https://твой-проект.vercel.app`.
   Отправь её другу — теперь вы оба можете открыть сайт и сыграть,
   находясь в любой точке мира.

## Локальный запуск (по желанию, перед деплоем)

```bash
npm install
# создай файл .env.local с переменными из Upstash:
# UPSTASH_REDIS_REST_URL=...
# UPSTASH_REDIS_REST_TOKEN=...
npm run dev
```

Открой http://localhost:3000
